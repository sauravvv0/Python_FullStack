Div Element

The <div> element is used as a container for other block-level elements. It is a block-level element itself.

This is a div container

The <div> element can contain multiple block-level elements such as paragraphs, headings, or even other <div> elements.

Block-Level Elements

Block-level elements take up the full width available and always start on a new line.

This is a Block-Level Element

This paragraph is a block-level element, so it starts on a new line and takes up the full width of its container.

This is another paragraph, which is also block-level. It starts on a new line below the heading.

Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary. Examples of inline elements include <span>, <a>, and <strong>.

This is a span element inside a paragraph. It does not start on a new line.

You can also have inline elements like links or bold text within paragraphs, and they will not break the flow of the content.
